* Broadcom iProc MDIO bus controller

Required properties:
- compatible: "brcm,iproc-mdio"
- reg: address and length of the regsiter set for the MDIO interface
- #size-cells: must be 1
- #address-cells: must be 0

Example:

mdio@0x18002000 {
	compatible = "brcm,iproc-mdio";
	reg = <0x18002000 0x8>;
	#size-cells = <1>;
	#address-cells = <0>;

	enet-gphy@0 {
		reg = <0>;
	};
};
