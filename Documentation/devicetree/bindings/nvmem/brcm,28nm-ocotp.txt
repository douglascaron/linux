Broadcom 28NM OTP memory controller

Required Properties:
- compatible: Must be "brcm,28nm-ocotp"
- reg: Base address of the OTP controller.
- brcm,ocotp-size: Amount of memory available, in 32 bit words

Example:

otp: otp@0301c800 {
	compatible = "brcm,28nm-ocotp";
	reg = <0x0301c800 0x2c>;
	brcm,ocotp-size = <2048>;
};
